== Description ==

Released July 2017 - Manage, list and sell cars and vehicles online quickly.

Responsive, light and very easy to install and manage. StartUp guide online available at plugin settings page.

You can find the <a href="http://cardealerplugin.com/demo/" target="_self">demo page here</a>
and <a href="http://cardealerplugin.com/demo-video/" target="_self">demo video here</a>

== Installation ==


1) Install via wordpress.org

2) Activate the plugin through the 'Plugins' menu in WordPress

or

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
